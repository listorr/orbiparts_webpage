🎯 LO QUE HACE EL SISTEMA:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  LOGIN AUTOMÁTICO
    ✓ Inicia sesión en SkyMart con tus credenciales
    ✓ Mantiene la sesión activa durante el scraping

2️⃣  BÚSQUEDA INTELIGENTE
    ✓ Busca productos de: Eastman, AeroShell, Castrol
    ✓ Navega por todas las páginas de resultados
    ✓ Extrae URLs de todos los productos encontrados

3️⃣  EXTRACCIÓN COMPLETA
    Para cada producto obtiene:
    ✓ Especificaciones (NSN, Part Type, Units, etc.)
    ✓ Precios y disponibilidad
    ✓ Inventario por almacén
    ✓ Shelf life y hazmat info
    ✓ Productos alternativos
    ✓ Links a datasheets

4️⃣  DESCARGA DE DOCUMENTOS
    ✓ PDFs de Safety Data Sheets (SDS)
    ✓ Datasheets técnicos
    ✓ Organiza en carpeta datasheets/

5️⃣  CONVERSIÓN A MARKETPLACE
    ✓ Formatea datos para tu marketplace
    ✓ Asigna imágenes apropiadas por tipo
    ✓ Genera ratings realistas
    ✓ Marca productos destacados
    ✓ Crea componente React listo para usar

6️⃣  ESTADÍSTICAS
    ✓ Resumen visual de productos
    ✓ Análisis por fabricante
    ✓ Distribución por tipo
    ✓ Almacenes y disponibilidad


🚀 CÓMO USARLO:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASO 1: Instalar dependencias
────────────────────────────────────────────────
cd tools
npm install


PASO 2: Configurar credenciales
────────────────────────────────────────────────
Edita: tools/skymart-scraper.js

Líneas 19-20:
  USERNAME: 'TU_EMAIL@SKYMART.COM',      ← CAMBIA ESTO
  PASSWORD: 'TU_CONTRASEÑA_REAL',        ← CAMBIA ESTO


PASO 3: Ejecutar el pipeline completo
────────────────────────────────────────────────
npm run full-pipeline

Esto ejecuta:
  ✓ Scraping de productos
  ✓ Descarga de datasheets
  ✓ Importación al formato marketplace
  ✓ Visualización de estadísticas


PASO 4 (OPCIONAL): Comandos individuales
────────────────────────────────────────────────
npm run scrape              # Solo scrapear
npm run view                # Ver estadísticas
npm run import              # Importar datos

npm run scrape:eastman      # Solo Eastman
npm run scrape:aeroshell    # Solo AeroShell
npm run scrape:castrol      # Solo Castrol

⚙️ CONFIGURACIÓN OPCIONAL:

MAX_PRODUCTS: 100          → Cantidad máxima por marca
DELAY: 2000                → Tiempo entre requests (ms)
HEADLESS: false            → Ver navegador en acción (debug)

🐛 SOLUCIÓN DE PROBLEMAS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Error de login
   → Verifica credenciales en CONFIG
   → Intenta con HEADLESS: false
   → SkyMart puede tener CAPTCHA (resuelve manualmente)

❌ No encuentra productos
   → Los selectores CSS pueden haber cambiado
   → Revisa la estructura HTML de SkyMart

❌ PDFs no se descargan
   → Verifica permisos de escritura
   → Algunos PDFs requieren auth adicional

❌ Puppeteer no funciona
   → cd tools && rm -rf node_modules && npm install
